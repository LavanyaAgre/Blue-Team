# AI-Powered Security Operations Lab

## Objective
Simulate real-world SOC tasks including EDR alert handling, firewall monitoring, and Zero Trust access control.

## Features
- EDR Alert Analysis (brute force, suspicious process)
- Firewall Rule Monitoring & anomaly detection
- Zero Trust access validation
- Basic incident classification

## Skills Demonstrated
- Threat Investigation
- Alert Triage
- Firewall Analysis
- Security Operations Workflow
- Incident Response Thinking

## Value
This project simulates how a Security Operations analyst handles alerts, investigates threats, and enforces access controls in a real environment.
